# Belize City First Church of the Nazarene Asset Database

This is a deployment-ready Next.js web app for a church asset database with:
- Login screen
- Role-based access
- Asset register
- User roles view
- Permissions view
- Audit log

## Demo Accounts

- worship@bcfcn.org / worship123
- finance@bcfcn.org / finance123
- viewer@bcfcn.org / viewer123

## Run locally
```bash
npm install
npm run dev
```

## Deploy to Vercel
1. Upload this folder to GitHub.
2. Sign in to Vercel.
3. Import the GitHub repository.
4. Click Deploy.
